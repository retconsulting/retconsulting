Questions fréquentes

Où sont stockées mes données ?

Uniquement sur votre appareil. dueday fonctionne 100 % hors-ligne et ne collecte aucune donnée. Voir la politique de confidentialité.

Faut-il créer un compte ?

Non. Aucune inscription, aucun email.

Comment sauvegarder mes données ?

Réglages, puis Données, puis Exporter : un fichier JSON est créé là où vous le choisissez (Fichiers, iCloud Drive…). Importez-le sur un nouvel appareil via Réglages, Données, Importer.

Les rappels ne s'affichent pas ?

Vérifiez Réglages iOS, Notifications, dueday, puis dans l'app Réglages, Notifications. Les paiements d'un même jour sont regroupés en une seule notification.

Comment supprimer toutes mes données ?

Réglages, Données, Tout réinitialiser. Ou désinstallez simplement l'app : tout disparaît avec elle.

Un conseil ?

Pensez à exporter une sauvegarde JSON régulièrement, notamment avant de changer d'appareil. L'application est fournie en l'état.

L'app est-elle vraiment gratuite ?

Oui. Toutes les fonctions essentielles sont gratuites, sans limite de durée. Seules d'éventuelles options purement cosmétiques pourraient un jour être payantes.

Le widget ne se met pas à jour ?

Ouvrez l'app une fois (le widget se synchronise à l'ouverture), ou retirez puis ré-ajoutez le widget.
