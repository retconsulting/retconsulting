Confidentialité

Ce site est volontairement minimal. Il ne dépose aucun cookie, n'utilise aucune mesure d'audience et n'intègre aucun traceur ni script tiers. Rien de votre visite n'est conservé dans votre navigateur, et les polices sont servies depuis ce site : votre navigateur n'envoie donc aucune requête à une société extérieure pendant votre lecture.

Le formulaire de contact n'envoie rien de lui-même. Il ouvre un message pré-rempli dans votre propre application de messagerie, que vous choisissez ensuite d'envoyer ou non. Lorsque vous m'écrivez, je n'utilise votre adresse et votre message que pour vous répondre, et ne les conserve pas plus longtemps que nécessaire.

Le site est hébergé sur GitHub Pages (GitHub, Inc.). Comme tout hébergeur web, GitHub peut journaliser des données techniques telles que votre adresse IP et votre navigateur afin de servir les pages et d'en assurer la sécurité. Ce traitement relève de la déclaration de confidentialité de GitHub.

Pour toute question sur vos données, ou pour exercer vos droits au titre du RGPD, écrivez à retbaz@retconsulting.be. Vous pouvez également introduire une réclamation auprès de l'Autorité de protection des données belge.
